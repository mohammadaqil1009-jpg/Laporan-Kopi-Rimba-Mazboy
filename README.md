# Laporan-Kopi-Rimba-Mazboy
Sistem laporan mini pos yang mudah digunakan dengan data detail
(CARA PENGGUANAAN)
Langkah awal dengan mengisi data absensi, untuk mengkonfirmasi kehadiran staff yang bekerja.
Kemudian input tiap jenis kopi yang terjual, maka jumlah cup dan total pendapatan (omzet) dapat terhitung otomatis.
Jika sudah menginput data jenis kopi yang terjual, jangan lupa untuk mengisi kolom pengeluaran dan pembayaran via qris.
Jika kedua kolom sudah di isi maka sistem akan otomatis menghitung total setor tunai yang harus kamu setorkan kepada atasan.
Jika sudah langkah terakhir staff tinggal hanya mengklik kirim laporan agar laporan data pejualan terkirim.
